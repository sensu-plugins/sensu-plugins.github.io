Fully leverage Sensu!

We have open source plugins, handlers and mutators to maximize the effective use of Sensu.

Popular plugins: CPU, Memory, Disk, and Network checks, system load, hardware checks, Redis, ElasticSearch, MySQL, Mesos, a mailer, PagerDuty integration, metric export to Graphite, AWS, HTTP requests and many more...

About Sensu Plugins

This project contains plugins, handlers, mutators and other code to maximize the effective use of Sensu in various types of auto-scaling and traditional environments. Most of them are implemented in Ruby and use the sensu-plugin gem.
